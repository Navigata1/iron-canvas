# Phase 3: COLLECT — Find Matching References

> Find inspiration that matches THIS site's unique personality — not a universal mood board.

## The Rule

Every site gets its OWN references. Never reuse the same mood board for different projects. The references must match the Feel Profile from Phase 2.

## Process

### 1. Identify the Industry Vertical
Based on the site's content and Feel Profile, what world does it live in?
- Luxury goods / beauty / fragrance
- Faith / nonprofit / community
- Education / academy / youth
- SaaS / technology / startup
- Food / restaurant / hospitality
- E-commerce / retail / DTC
- Healthcare / wellness
- Creative / portfolio / agency

### 2. Find 3-5 Best-in-Class References
Search for premium sites in that vertical. Look for:
- Sites that evoke the SAME emotions as your Feel Profile
- Sites with similar palette temperature (warm/cool)
- Sites with comparable content density
- Sites that make you think "this is what ours should feel like"

### 3. Screenshot Specific Elements
Don't screenshot entire pages. Capture the specific things you want to learn from:
- A hover state that feels premium
- A scroll animation that adds depth
- A typography treatment that creates hierarchy
- A product display that makes you want to buy
- A CTA section that drives action
- A loading sequence that feels polished

### 4. Document What Makes Each Premium
For each reference, note:
```
REFERENCE: [URL]
WHAT I'M TAKING: [specific element or technique]
WHY IT FITS: [how it matches our Feel Profile]
ADAPTATION NEEDED: [how to make it work with our DNA]
```

### 5. Create a Fit Matrix

| Reference Element | Matches Our Palette? | Matches Our Feel? | Adaptation Needed |
|------------------|---------------------|-------------------|-------------------|
| Hover glow effect from [site] | ✅ Can use gold glow | ✅ Premium feel | Change blue to gold |
| Scroll parallax from [site] | ✅ Works with any palette | ✅ Adds depth | Adjust speed |
| Card layout from [site] | ❌ Too minimal for our content | ⚠️ Partial match | Skip or heavily modify |

## What NOT To Do

- ❌ Use Awwwards/Dribbble as your only source (most are style-over-substance)
- ❌ Pick references because they look cool regardless of fit
- ❌ Copy layouts verbatim — adapt techniques, don't clone pages
- ❌ Collect more than 5 references (diminishing returns, decision paralysis)

## Phase 3 Completion Criteria

- [ ] Industry vertical identified
- [ ] 3-5 reference sites found that match the Feel Profile
- [ ] Specific elements screenshotted (not full pages)
- [ ] Each reference documented with what/why/adaptation notes
- [ ] Fit matrix completed
- [ ] No references that contradict the Feel Profile's anti-feelings
