# 🎨 Iron Canvas v3

**Adaptive Site-Aware Multi-Agent Design Orchestration**

Iron Canvas is a methodology for making any digital experience the best version of ITSELF. It uses a 7-phase pipeline and a multi-agent swarm architecture to produce premium, site-aware results.

## monorepo Structure

```
iron-canvas/
├── Iron_Canvas_v3.md          ← Master methodology & core principles

├── phases/                    ← Phase-by-phase actionable instructions

```

## Quick Start

1. **Phase 0:** Orient the project and select execution mode (SOLO/SWARM/MISSION).
2. **Phase 1:** STUDY the existing site DNA. **Do not skip this.**
3. **Phase 3.9:** Generate the Design PRD.
4. **Dispatch:** Use `sessions_spawn` to deploy parallel build agents.
