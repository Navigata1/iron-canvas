Three sites enhanced by Iron Canvas look like three DIFFERENT premium sites — not three copies of the same dark template.

## Three Execution Modes

| Mode | Agents | Speed | Best For |
|------|--------|-------|----------|
| **SOLO** | 1 agent, sequential | Baseline | Quick enhancements, small sites |
| **SWARM** | Orchestrator + 4 parallel Build Agents | 2-3× faster | Standard production |
| **MISSION** | Full 8-agent team | Maximum quality | Complex sites, dashboards, flagship work |

## The 10-Phase Pipeline

```
ORIENT(0) → STUDY(1) → FEEL(2) → SCOUT(3) → VALIDATE(3.5) → PACKAGE(3.9)
→ [PARALLEL BUILD AGENTS] → COMPOSE(6) → VERIFY(7)
```

| # | Phase | Output |
|---|-------|--------|
| 0 | **Orient** | Project type + execution mode |
| 1 | **Study** | `site-dna-profile.json` |
| 2 | **Feel** | `feel-profile.json` + Brand Personality Matrix |
| 3 | **Scout** | `north-star-reference.png` + `scout-report.json` |
| 3.5 | **Validate** | `technical-validation-report.md` |
| 3.9 | **Package** | `design-prd.md` |
| 4-5 | **Forge + Generate** | `motion-blueprint.json` + artifacts |
| 6 | **Compose** | Complete implementation |
| 7 | **Verify** | `verification-report.json` + final verdict |
