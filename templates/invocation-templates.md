# Invocation Templates

> Copy-paste these to invoke Iron Canvas for different workflows.

## Full Enhancement (Existing Site)

```
Apply Iron Canvas to [URL]. Start with Phase 1: Study the site's DNA. 
Do not change anything until you've profiled the palette, fonts, imagery, and personality.
Work through all 7 phases. Generate artifacts with Nano Banana Pro, 
verify each in context before proceeding.
```

## Full Enhancement + Scroll Engine

```
Apply Iron Canvas to [URL] with scroll-driven product reveal.
Phase 1 first (DNA Profile), then work through all phases.
Build the scroll engine with GSAP ScrollTrigger + HTML5 Canvas.
Generate keyframes with Nano Banana Pro. Content overlays sync with scroll progress.
```

## New Site from Scratch

```
Build [project name] using Iron Canvas. Start with Phase 2: Define the feel.
What should users feel when they use this? 
Then proceed through Phases 3-7 with artifact generation via Nano Banana Pro.
```

## Artifact Generation Only

```
Iron Canvas Phase 5: Generate artifacts for [site/project].
Study the site's palette and context first (Phase 1 DNA Profile required).
Show me 4 variants of each artifact before selecting. Test each in situ.
```

## Scroll Sequence Only

```
Iron Canvas scroll sequence for [site/project].
Generate [N] keyframes with Nano Banana Pro using the consistency protocol.
Build GSAP ScrollTrigger + Canvas scroll engine.
Content overlays at [describe what text appears when].
```

## Phase-Specific Invocation

```
Iron Canvas Phase [N] for [site/project].
[Phase 1: Study] Complete DNA Profile.
[Phase 2: Feel] Discover emotional target.
[Phase 3: Collect] Find matching references.
[Phase 4: Forge] Structural enhancement.
[Phase 5: Generate] Context-aware artifacts.
[Phase 6: Compose] Integration + scroll engine.
[Phase 7: Refine] Iterative polish.
```

## Quick Enhancement (Skip Scroll Engine)

```
Apply Iron Canvas Phases 1-5 to [URL]. 
No scroll engine needed — just enhance the existing design with 
better interactions, context-aware artifacts, and premium polish.
```
